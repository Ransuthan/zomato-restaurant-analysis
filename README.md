# Zomato Restaurant Data Analysis

Exploratory Data Analysis (EDA) on the Zomato restaurant dataset
containing 9,551 restaurants across multiple countries.

## About the Project

This project analyzes the Zomato restaurant dataset to uncover
patterns in restaurant ratings, cuisines, pricing, and online
delivery adoption across different cities and countries.

## Tools Used

- Python 3.11
- Pandas — data cleaning and analysis
- Matplotlib — data visualization
- Seaborn — statistical charts
- Jupyter Notebook (VS Code)

## Dataset

- 9,551 restaurants
- 21 columns including rating, cuisines, cost, location, delivery info
- Countries: India, USA, UK, UAE, and more

## Key Findings

- India dominates the dataset with the highest number of restaurants
- North Indian cuisine is the most popular cuisine overall
- Restaurants with online delivery have a slightly higher average rating
- Higher price range restaurants tend to receive better ratings
- Only a small percentage of restaurants offer table booking
